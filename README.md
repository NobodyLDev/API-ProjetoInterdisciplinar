**Projeto P.I - API de Gerenciamento de Materiais e Produção**

**Descrição do Projeto**

Este projeto tem como objetivo desenvolver uma API para gerenciamento de materiais e controle de produção. A aplicação permitirá cadastrar materiais, definir produtos com base nesses materiais e realizar simulações de produção, facilitando a tomada de decisões e o controle de estoque.
A API foi pensada para atender cenários como produção sob demanda, controle de insumos e planejamento de fabricação.

**Tecnologias** 
Node.js
TypeScript
Banco de Dados 

**Observações**

Este projeto está sendo desenvolvido como parte de um Projeto Integrador (P.I), com foco em aprendizado prático e aplicação de conceitos de desenvolvimento backend.
